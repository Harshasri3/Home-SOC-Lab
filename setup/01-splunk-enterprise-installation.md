# Splunk Enterprise Installation

## Objective

Install Splunk Enterprise and configure it as the SIEM platform for the Home SOC Lab.

## Installation

- Installed Splunk Enterprise (Windows 64-bit)
- Installed using the Local System account
- Configured the administrator account
- Verified the Splunk Web interface

## Verification

- Successfully accessed Splunk Web at http://localhost:8000
- Logged in as the administrator
- Confirmed the Search & Reporting application is available

Splunk Enterprise was successfully installed and is ready to receive and analyze Windows event logs.
